# ai-strategies.org

Static two-page website for an AI strategy consultancy targeting the DACH region.

## Structure
- `index.html` — English landing page
- `dach.html` — German DACH page (primary content)
- `styles.css` — Shared stylesheet

## Tech
- Plain HTML + CSS, no JavaScript, no build step
- Designed for static hosting (GitHub Pages, Netlify, etc.)

## Design
- Minimal & professional, typography-driven
- System font stack with Inter preferred
- Max-width 720px content container
- Responsive via fluid design + one mobile breakpoint
